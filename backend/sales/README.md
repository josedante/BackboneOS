# Aplicación `sales`

La app `sales` define modelos para oportunidades comerciales, sesiones de venta y listas de contacto.

**Estado:** la app **no está incluida en `INSTALLED_APPS`**. El código existe en el repositorio pero no está activo en el runtime actual.

**Integración con touchpoints (planificado):** ver la sección [Captura contextual: integración sales (planificado)](../../docs/backend/connectors.md#captura-contextual-integración-sales-planificado) en [docs/backend/connectors.md](../../docs/backend/connectors.md).
